Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Tyles of lists:

  • unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute Go to google.com to search for more info.

you have to type in https:// for the links

Setting 'target="blank"' will open the link in a new window. See MDN for more details

Anchor tags can also create links to a new location on the same page. Instead of putting a URL in the "href" attribute, just use the "is" of the element you want to navigate to. This allows you to Just to the top of the page.

  1. Add anchor tag to the location that you want to go to. In this case, it is top of the code in h1
  2. Now come back to the location that you want the link
  3. Create an anchor tag with '< a'
  4. Add in href. So now it is '< a href = "#id">' Make sure to add # in front of the id.
  5. Then add in the text that you want to be shown in the link
  6. Finally, close it off

Text Emphasis

Italic

The 'em' element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold in most browsers.
